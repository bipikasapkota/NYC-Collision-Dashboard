# NYC Motor Vehicle Collision Dashboard
PCDA - Final Project by Pooja Mengaji &amp; Bipika Sapkota

## Overview
This project analyzes motor vehicle collision data in New York City using Python and Streamlit. The dashboard provides insights into crash patterns across time, location, and severity.

## Features
- Crash trends by year, month, and hour
- Location-based analysis (borough-level)
- Injury severity analysis
- Heatmap of injury rates by time and borough
- Interactive Streamlit dashboard

## Tools Used
- Python
- Pandas
- Plotly
- Seaborn
- Streamlit

## How to Run
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt

3. Run the app:
   streamlit run app.py

## Dataset
Motor Vehicle Collision dataset
